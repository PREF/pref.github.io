Powerful hex editing

With PREF you can analyze and edit unknown binaries easily with the help of 010 editor template language

Visual Reversing

You can inspect displayed data with dotplot chart and check for known patterns

Charts

Byte occurrences and entropy charts are available in order to understand encrypted/packed binaries

File Format analysis

Open Source

Source code is GPL3 licensed and hosted on github, feel free to fork it and send patches.

Cross Platform

The UI is just a frontend over PrefLib, you can create your own with any other toolkit for every platform.

Scriptable

You can automate hex editing or parse file formats with 010 Editor compatible language.

Visual Reversing. Read bytes as pixel.

The Visual Reversing approach helps you to understand unknown data easily, currently PREF provides only DotPlot view, more will be added in the future.
Histogram and Entropy charts are also available.

Advanced scripting

PREF aims to be compatible with 010 Editor scripts: in the future Capstone and Keystone engines will be integrated for simple instruction analysis.

Qt Powered

PREF's UI is written in Qt5 in order to bring the best possible experience in every platform.
